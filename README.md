##  🎮 Como usar

Faça um clone deste repositório:

```bash
  https://github.com/deciofrancis/dogs.git
```
Instale as dependências:
```bash
  # com npm
  $ npm install

  # com yarn
  $ yarn install
```
Inicie o projeto com o comando:animais
```bash
  # com npm
  $ npm run dev

  # com yarn
  $ yarn run dev
```

Acesse o projeto abrindo localhost:porta no seu navegador.
